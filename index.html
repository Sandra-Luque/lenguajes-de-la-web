    Hoja de estilos en cascada o CSS (siglas en ingles de cascading style sheets) es un lenguaje usado para definir y crear la presentacion de un documento estructurado escrito en HTML

    JavaScript (abreviado comunmente "Js") es un lenguaje de programacion interpretado utilizado principalmente en el lado del cliente (client-side) implementado como parte de un anvegador web permitiendo mejoras en la interfaz de usuario y paginas web dinamicas.
